================================================================================
RESPONSIVE DESIGN
================================================================================

MOBILE (< 640px):
- Sidebars hidden, toggle with hamburger menu ☰
- Cards stack vertically
- Buttons full width (48px min height)
- Single column layout
- Larger fonts for readability

TABLET (640px - 1024px):
- Sidebar drawer (collapsible)
- 2 column grid for cards
- Touch-friendly spacing
- Medium screen optimized

DESKTOP (> 1024px):
- Permanent sidebar
- Multi-column layouts
- Full feature set visible
- Optimized for productivity

================================================================================
INTERACTIVE STATES
================================================================================

BUTTON STATES:
Default    → Indigo background, white text
Hover      → Darker indigo + shadow increase
Active      → Darkest indigo + shadow decrease
Loading    → Spinner animation
Disabled   → Gray + 50% opacity

FORM INPUTS:
Default    → Gray border
Focused    → Indigo border + glow shadow
Filled     → Dark text value shown
Error      → Red border + error message
Success    → Green checkmark

CARDS:
Default    → Subtle shadow
Hover      → Increased shadow + slight lift
Selected   → Indigo border highlight

================================================================================
DARK MODE
================================================================================

Automatically applies:
- White text on dark backgrounds
- Dark gray (#1F2937) to near black (#0F172A)
- Lighter gray borders (#374151)
- Dark gray cards (#111827)
- Adjusted shadows and glows
- Toggle in navbar (top-right)

================================================================================
ANIMATIONS
================================================================================

Page Load       → Fade in + slide down (200ms)
Modal Open      → Scale from center (300ms)
Toast Show      → Slide in from bottom (200ms)
Button Hover    → Scale 1.02x + shadow (200ms)
Loading         → Spinner rotation (infinite)
Success         → Checkmark animate (300ms)
